# Telegram Random Number Bot

Бот-рандомайзер чисел с функциями:
- Генерация случайных чисел в заданном диапазоне
- Режим "один раз" - числа не повторяются
- Исключение определенных чисел
- История генераций

## Установка

1. Клонируй репозиторий
2. Установи зависимости: `pip install -r requirements.txt`
3. Создай файл `.env` и укажи BOT_TOKEN
4. Запусти: `python -m bot.main`
